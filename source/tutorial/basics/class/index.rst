Python类和对象
==============

Python语言在设计之初, 就定位为一门面向对象的编程语言, "Python中一切皆对象"就是对Python这门编程语言的完美诠释.

类和对象是Python的重要特征, 相比其它面向对象语言, Python很容易就可以创建出一个类和对象. 
同时, Python也支持面向对象的三大特征: 封装, 继承和多态.
