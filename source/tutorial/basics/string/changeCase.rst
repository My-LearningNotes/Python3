字符串大小写转换
================


``title()``\ 方法
-----------------

``title()``\ 方法用于将字符串中每个单词的首字母转换为大写, 其它字母全部转换为小写, 此方法会返回转换后得到的字符串. 
如果字符串中没有需要被转换的字符, 此方法将字符串原封不动地返回.

.. code-block:: python

    string.title()


``lower()``\ 方法
-----------------

``lower()``\ 方法用于将字符串中的所有大写字母转换为小写字母, 转换完成后, 该方法返回新得到的字符串. 

.. code-block:: python

    string.lower()


``upper()``\ 方法
-----------------

``upper()``\ 方法的功能和\ ``lower()``\ 方法恰好相反, 它用于将字符串中的所有小写字母转换为大写字母, 返回转换后的字符串.

.. code-block:: python

    string.upper()
