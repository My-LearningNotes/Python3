Python字符串常用方法
====================

在Python中, 字符串是一种内置的基本数据类型.

在开发过程中, 经常需要对字符串进行一些特殊处理, 比如拼接字符串, 截取字符串, 格式化字符串等, 
这些操作无需开发者自己设计实现, 只需要调用相应的字符串方法即可.
